# ShopVerse - E-Commerce Product Catalog

## Overview

ShopVerse is a modern and responsive E-Commerce Product Catalog web application built using HTML, CSS, and JavaScript.

The project demonstrates frontend architecture, responsive UI design, product filtering, and deployment workflows.

---

## Features

- Responsive Design
- Product Cards
- Search Functionality
- Add to Cart Counter
- Smooth Navigation
- Modern UI/UX
- Mobile Friendly Layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Git & GitHub
- Vercel Deployment

---

Deploy easily using:
